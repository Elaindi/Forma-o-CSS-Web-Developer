🎨 Formação CSS Web Developer

Repositório dedicado ao aprendizado de CSS, com conteúdos organizados em módulos práticos que vão do básico ao avançado na estilização web.

📚 Sobre o projeto

Este projeto reúne exercícios, exemplos e práticas voltadas ao estudo de CSS, com foco em:

Estilização de elementos
Manipulação de layout
Tipografia
Responsividade

Cada pasta representa um tema específico, facilitando o aprendizado de forma progressiva e organizada.
