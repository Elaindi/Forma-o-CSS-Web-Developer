# Formação CSS Web Developer

Repositório de estudos e exercícios de CSS organizados em módulos práticos para quem está aprendendo ou revisando conceitos de estilização web.

## Sobre o Projeto

Este repositório reúne exemplos, exercícios e projetos orientados ao aprendizado de CSS, com foco em:

- Estilização de elementos
- Manipulação de layout
- Tipografia
- Responsividade
- Boas práticas de organização de código

Cada pasta contém um tema específico para estudo progressivo.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Conteúdos Abordados

- Bordas: tipos, cores, arredondamento e border-image
- Tipografia: fontes personalizadas, tamanho de fonte, letter-spacing e word-spacing
- Texto: text-transform, text-align e text-decoration
- Layout e display: display block, display inline, overflow e box-shadow
- Exercícios práticos para reforçar os conceitos aprendidos

## Objetivos

- Consolidar conhecimentos em CSS
- Praticar propriedades essenciais
- Melhorar a organização de código
- Criar uma base sólida para desenvolvimento web
