Seletores CSS

Bem-vindos!

No CSS nós possuímos vários tipos de seletores:

  • Seletor por tipo: busca elementos por tags HTML específicas (ex: p, h1).
  • Seletor por ID (#): busca elementos através do atributo id.
  • Seletor por classe (.): busca elementos através do atributo class.
  • Seletor universal (*): seleciona todos os elementos HTML.
  • Seletor de atributo ([atrib]): seleciona elementos que possuem um atributo específico (ex: [type="text"]).
